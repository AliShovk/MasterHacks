# Чеклист передачи проекта

## Что должно входить в архив передачи

- исходный код проекта
- `.env.example`
- SQL-дамп
- инструкция для покупателя
- карточка проекта для продажи

## Что меняется под каждого покупателя

- домен
- SSL
- Telegram bot token
- Telegram username
- ссылка на бота
- параметры базы данных
- `ADMIN_PANEL_KEY`
- логотип и брендовые материалы
- контакты поддержки

## Технический чеклист запуска

- проект загружен на сервер
- файл `.env` создан на основе `.env.example`
- база данных создана
- SQL импортирован
- PHP работает
- веб-сервер настроен
- HTTPS работает
- webhook установлен
- Telegram-загрузка протестирована
- модерация протестирована
- лента открывается корректно

## Премиальная передача под ключ

Дополнительно можно включать в сделку:

- настройку домена
- выпуск SSL сертификата
- создание Telegram-бота через BotFather
- установку webhook
- замену бренда
- запуск на сервере клиента
- сопровождение после передачи
